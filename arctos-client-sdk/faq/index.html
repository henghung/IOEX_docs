    常見問題¶

    以下列出關於 Arctos Client SDK 相關的常見問題與回答。

    Arctos Client SDK 由誰開發及維護？¶

    Arctos Client SDK 為 桓竑智聯股份有限公司 所開發及維護的互動影音通訊解決方案。

    Arctos Client 只能跟 Switch 通訊嗎？能讓 Client 之間彼此通訊嗎？¶

    是的，Arctos Client 只能跟 Switch 通訊。Arctos Client SDK 的通訊架構是以 Switch 為中心，由 Switch 整合多個 Client 的影音後，發送給所有 Client。

    是否可以使用模擬器進行開發？¶

    可以，但請注意以下說明。

    無論是 Arctos Client SDK 或是 Arctos Switch，都需要 Google Play Service。因此必須要在有安裝 Google Play Service 的模擬器環境上進行開發。

    另外，通常模擬器的計算效能不如實體機器，因此在影像傳輸過程中，可能會因為頻繁的影音編碼、解碼計算造成模擬器負擔過重，導致畫面出現停頓、破格的現象。

    Arctos 使用的傳輸密碼技術是什麼？影音編碼技術是什麼？¶

    Arctos SDK 使用 xsalsa20 對稱式密碼系統加密訊息，使用 curve25519 公開金鑰系統進行金鑰交換。金鑰長度為 256 位元。

    影像編碼採用 vp8 格式，音訊編碼採用 opus。
