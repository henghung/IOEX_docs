# IOEX Documents

HTML documents of IOEX SDK. The documents can be access via https://henghung.github.io

## Usage

This repository utilizes [Github Pages](https://pages.github.com/) feature. You can access the HTML documents in the repository via default github pages domain https://henghung.gitbub.io/<SDK type>/<version code>.

For example, to visit IOEX Core SDK version 0.14.0-rc1, you can use the following url:
https://henghung.github.io/core/0.14.0-rc1
